## Pandas Data Analysis Project – Retail Sales Dataset

I recently completed a data analysis project using Pandas, where I worked with a retail sales dataset containing over 2000 transaction records.

The objective of this project was to practice data cleaning, manipulation, and analysis using Pandas, one of the most important libraries used in data science and analytics.

## Key tasks performed in this project:

* Data loading and exploration using Pandas
* Handling missing values and duplicate records
* Cleaning inconsistent data formats
* Data filtering and sorting
* Feature creation and data manipulation
* Aggregation using groupby()
* Frequency analysis using value_counts()
* Advanced analysis using pivot tables
* Extracting business insights from the dataset
